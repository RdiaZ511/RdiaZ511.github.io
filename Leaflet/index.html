Carabobo Leaflet

Guía educativa completa del uso de Leaflet.js enfocada en el corazón de Venezuela.

L.map

El núcleo de la aplicación. Crea un mapa en un div de HTML con coordenadas y zoom inicial.

const map = L.map('map').setView([10.1, -68.0], 10);

L.tileLayer

Define la fuente de las imágenes del mapa (tiles). Aquí usamos OpenStreetMap.

L.tileLayer('https://tile.openstreetmap.org/{z}/{x}/{y}.png').addTo(map);

L.marker

Añade marcadores icónicos en coordenadas específicas. Ideales para puntos de interés.

L.marker([10.16, -67.99]).addTo(map);

L.popup

Ventanas emergentes que aparecen al hacer clic en un objeto. Muy útiles para mostrar información extra.

marker.bindPopup('¡Bienvenido a Valencia!');

L.tooltip

Etiquetas pequeñas flotantes que aparecen al pasar el mouse por encima del elemento.

marker.bindTooltip('Valencia', { permanent: false });

L.circle

Dibuja un círculo sobre el mapa. Requiere centro y radio en metros. Útil para expresar áreas de influencia.

L.circle([10.2, -67.9], { radius: 500 }).addTo(map);

L.polygon

Permite dibujar formas geométricas cerradas basándose en una lista de coordenadas (lat, lng).

L.polygon([[lat1, lng1], [lat2, lng2], ...]).addTo(map);

L.polyline

Dibuja líneas que conectan varios puntos. Ideal para rutas de transporte o fronteras.

L.polyline([[10.1, -68.0], [10.2, -68.1]]).addTo(map);

L.layerGroup

Permite agrupar varios elementos (marcadores, círculos) para controlarlos como una sola unidad.

const group = L.layerGroup([marker1, marker2]).addTo(map);

L.control.layers

Control de capas que permite al usuario alternar entre diferentes mapas base y superposiciones.

L.control.layers(baseMaps, overlays).addTo(map);

L.control.scale

Un control visual sencillo que indica la escala actual del mapa en kilómetros o millas.

L.control.scale().addTo(map);

Event Handling

Permite interactuar con el mapa. Detecta clics, movimientos de cámara o cambios de zoom.

map.on('click', (e) => console.log(e.latlng));
